# BK-EV-NIP
Balitsky Kovchegov evolution for non-impact-parameter dependent Dipoles at LO (including running couple and various prescriptions)

Please cite [XXX] when using this code.

## Requisites 

### GSL

This evolution code uses the GSL library for integration routines, as well as for the evaluation of special functions. The GSL library can be found [here](https://www.gnu.org/software/gsl/)


## How to compile

The compilation can be performed simply by using the command  

```
make
```
and to run

```
./compute
```
